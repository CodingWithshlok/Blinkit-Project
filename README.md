📊 Blinkit Sales & Outlet Performance Dashboard

An end-to-end Retail Data Analysis project built using Power BI to analyze sales performance, outlet trends, and product contribution across different location tiers.

This project demonstrates how raw retail data can be transformed into actionable business insights using Business Intelligence tools.

🚀 Project Objective

The goal of this project was to:

Analyze total sales performance

Compare outlet sizes and location tiers

Identify top-performing outlet types

Study product category contribution

Understand the impact of ratings and item visibility

Simulate real-world retail decision-making

📈 Key Performance Indicators

Total Sales: 35.38K

Average Sales: 140.95

Average Rating: 3.83

Total Items Analyzed: 251

These KPIs provide a quick overview of overall retail performance.

📍 Key Insights

Tier 3 outlets generated the highest revenue (14.03K), indicating strong demand in emerging markets.

Supermarket Type1 contributed 22.49K in total sales, making it the highest performing outlet category.

Medium-sized outlets showed consistent and stable growth.

Regular fat content items generated more revenue compared to low-fat products.

Higher item visibility positively influenced sales performance.

Sales trends show peak growth around 2018.
